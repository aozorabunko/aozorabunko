黒き素船

岩野泡鳴




黒き　素船（すぶね）　を　透（す）かし見れば、
砂　に　しやがめる　影　と　まがひ、
沖　の　小島　の　薄き　見れば、
人　の　思ひに　沈む　けはひ、
空　に　住へる　月　を　仰ぎ、
寂（さ）びし　わが身　の　魂（たま）　と　見たり。
われは　そのまま　まなこ　閉ぢて、
消ゆる　世界　を　今ぞ　いだく。

浮けよ、沈めよ、千々（ちぢ）の　なやみ、
千々の　悲（かなし）み、身をば　乗せて。
苦なるいのち　は――繁（しげ）き矢　なり――
積みて　重なる　夢　の　小船。
さして　行くへ　を　こゝに　問はじ、
この夜、この時　われは　活きん。
